# RGT Weekly Task

This repository contains the weekly tasks for RGT AI training.

## Overview

Each week, new tasks will be added to help improve and train AI models. These tasks will cover various aspects of AI development and training.

## Structure

The repository is organized as follows:

- `week-1/`: Contains tasks for the first week.
- `week-2/`: Contains tasks for the second week.
- `week-3/`: Contains tasks for the third week.
- `week-4/`: Contains tasks for the forth week.

- `...`
